# GarageOnline - Sitio Ecommerce de Venta de Vehículos

## Descripción del proyecto
GarageOnline es un sitio web ecommerce altamente interactivo para la venta de vehículos, que incluye un completo catálogo con filtros, detalles de producto con galería de imágenes, carrito de compras simulado, proceso de pago ficticio y generación de facturas en PDF con jsPDF. Además, cuenta con perfil de usuario y favoritos, y una página de contacto con mapa interactivo basado en Leaflet y OpenStreetMap.

El proyecto busca ofrecer una experiencia de usuario accesible, moderna, adaptativa y dinámica, implementada con tecnologías web estándar (HTML5, CSS3, JavaScript ES6+) y frameworks populares como Bootstrap y Font Awesome.

---

## Tecnologías utilizadas

- **HTML5**: Estructura semántica y accesible del sitio.
- **CSS3**: Diseño visual, estilos adaptativos y animaciones.
- **JavaScript (ES6+)**: Lógica para filtros, carrito, modales y generación de PDF.
- **Bootstrap 5**: Framework CSS para diseño responsive y componentes UI.
- **Font Awesome 6**: Iconografía vectorial moderna para interfaz.
- **jsPDF**: Generación de facturas en formato PDF en el cliente.
- **Leaflet**: Biblioteca para mapas interactivos con datos de OpenStreetMap.
- **LocalStorage**: Para persistencia local del perfil y favoritos.

## Instrucciones de despliegue

### Uso local con Visual Studio Code

1. Clona o descarga el proyecto.
2. Abre la carpeta en Visual Studio Code.
3. Instala la extensión **Live Server** desde el marketplace.
4. Abre el archivo `index.html`.
5. Haz clic derecho y selecciona **"Open with Live Server"**.
6. El sitio se abrirá en el navegador con recarga automática al guardar cambios.

### Publicación en GitHub Pages

1. Crea una cuenta en [GitHub](https://github.com).
2. Crea un nuevo repositorio público para el proyecto.
3. En la pestaña "Code", selecciona "Upload files".
4. Arrastra y suelta los archivos del proyecto (`index.html`, `style.css`, `script.js`, etc.).
5. Haz commit de los cambios.
6. Ve a **Settings > Pages**.
7. En "Source", selecciona la rama `main` y carpeta raíz `/root`.
8. Haz clic en "Save" y espera unos minutos.
9. Accede a la URL generada para ver el sitio en línea.

### Publicación en Vercel

1. Regístrate o inicia sesión en [Vercel](https://vercel.com).
2. Crea un nuevo proyecto vinculado a tu repositorio GitHub (opcional).
3. Si prefieres, realiza un deploy manual subiendo el código.
4. Vercel realizará el build automáticamente.
5. Obtén la URL pública generada para compartir el sitio.
